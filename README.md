# sales-insights
Designed and developed a comprehensive sales analysis dashboard in Power BI, visualizing key metrics such as revenue, sales quantity, and top-performing markets, customers, and products.
Extracted, transformed, and loaded (ETL) data from SQL databases, ensuring data integrity and accuracy throughout the process.
Utilized advanced Power BI functionalities to create dynamic visualizations and insightful reports, enabling data-driven decision-making.
Validated and cross-checked data accuracy by comparing dashboard metrics with raw SQL query results, ensuring reliability and consistency in the reported numbers.
